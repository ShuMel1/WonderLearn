# Word audio clips

Bundled pronunciation recordings, played by `DefaultPronouncer`.

- **Armenian only** for now (English/Russian use the device text-to-speech engine).
- File name: `hy_<wordId>.m4a` — e.g. `hy_apple.m4a`, `hy_dog.m4a`.
- `<wordId>` values come from `SeedData.kt` (apple, dog, red, …).
- If a clip is missing, the app falls back to "pronunciation coming soon".

Drop the recorded `hy_*.m4a` files directly into this folder.
